# BMC-01 Matrix First Read

## Purpose of this document

This document records the first internal reading of the initial **BMC-01 variants matrix** in the **Quantum–Spacetime Bridge** project.

Its purpose is not to produce a final interpretation, but to capture the first structured lessons from the comparative probe map across:

- `global_weight_permutation`
- `within_shell_weight_permutation`
- `within_local_group_weight_permutation`

combined with strength levels:

- `low`
- `medium`
- `high`

This is an internal reading note. It is meant to preserve the first interpretive structure before the project moves on to follow-up blocks or refined readouts.

## Status of the matrix result

The BMC-01 matrix is now more than a software smoke test.

The current matrix already supports a genuine structural comparison because:

- the intervention block runs successfully across multiple variants
- arrangement-sensitive readouts now react to weighted reassignment
- decision labels are no longer forced by readout blindness alone
- comparative response patterns are visible across intervention families

This does **not** yet constitute proof of carrier status. But it does justify a first internal read.

## Main comparative pattern

The current first-pass matrix pattern can be summarized as follows:

### 1. Global permutation produces the strongest overall response

Global weighted permutation shows the largest arrangement-sensitive response, especially at medium and high strength.

Observed pattern:
- low: weaker / borderline
- medium: strong response
- high: strong response

This is expected and methodologically useful. Global reassignment disrupts the weighted relational arrangement most aggressively while leaving the broad graph shell intact.

### 2. Within-shell permutation is comparatively more robust at lower strength

Within-shell permutation remains much less disruptive at low and medium strength, but becomes clearly stronger at high strength.

Observed pattern:
- low: modest response
- medium: still modest response
- high: stronger response

This is one of the most interesting results in the matrix.

### 3. Within-local-group permutation reacts earlier than expected

Within-local-group permutation already produces a clear response even at lower strength levels.

Observed pattern:
- low: clear nontrivial response
- medium: similarly clear response
- high: still clear response, but not dramatically larger

This suggests that local relational arrangement is already a sensitive carrier of structural change.

## First interpretive reading

The matrix supports a first qualitative hierarchy of disruption sensitivity.

### Highest disruption sensitivity
- `global_weight_permutation`

### Early local sensitivity
- `within_local_group_weight_permutation`

### Partial shell robustness
- `within_shell_weight_permutation` at low and medium strength

This does not yet define a final structural hierarchy, but it gives a first working map of where the current bridge-facing readout is most and least fragile.

## Why the within-shell result matters

The within-shell result is the most conceptually interesting current finding.

If weights can be reassigned **within the same shell** with only limited disruption at low and medium strength, then shell membership may already preserve a meaningful part of the relevant organization.

This suggests at least the following possibility:

> shell structure may not be merely a convenience label, but a partially order-bearing layer

This is still provisional. But it is now a serious interpretive option.

## First working interpretation of shell structure

The matrix result invites the following internal reading:

### Conservative version
Shell structure may act as a **robust intermediate organizational layer** that preserves part of the bridge-relevant arrangement even when exact pair-level weight assignment changes within the shell.

### Stronger version
Shell structure may itself be a **carrier of order** rather than only a partitioning device.

At the current stage, the conservative version is the safer internal statement. The stronger version remains a live candidate, not a conclusion.

## What the local-group result suggests

The local-group intervention is also highly informative.

Its relatively early response suggests that:

- local relational arrangement is meaningful
- bridge-relevant organization is not only a global arrangement effect
- fine local placement of weights matters
- the bridge candidate may depend on local support structure more strongly than previously visible

This strengthens the idea that the weighted relational layer is not just globally important, but internally structured into sublayers of varying sensitivity.

## Emerging picture of structural layering

The matrix suggests that the bridge candidate may not live in a single flat layer.

A plausible first layered interpretation is:

1. **topology** — too coarse to carry the current readable effect by itself  
2. **shell structure** — more informative and partially order-preserving  
3. **local weighted arrangement** — highly sensitive and likely bridge-relevant  
4. **full weighted relational configuration** — strongest current candidate location of bridge-bearing structure

This layered reading is preliminary, but it is already more informative than a simple “weights matter” statement.

## Marker–carrier relevance of the matrix

The matrix does not yet settle marker versus carrier.

However, it changes the balance of suspicion.

Before the matrix, the bridge candidate could still be read mainly as a strong marker with open carrier status.

After the matrix, the current position becomes sharper:

> arrangement-sensitive perturbations produce structured differential responses, which raises the carrier suspicion beyond the level of a purely diagnostic layer.

This is still not final carrier evidence. But it is no longer methodologically comfortable to treat the weighted relational layer as a mere passive readout surface.

## Why this is still not enough

Several reasons prevent over-reading.

### 1. Decision thresholds are still heuristic
The current `carrier_leaning` labels are scaffold-level decisions, not final inference rules.

### 2. The baseline is still synthetic
The current matrix uses a small template baseline, not yet a project-derived real baseline state.

### 3. Readout architecture is still early-stage
The arrangement-sensitive readouts are much better than before, but still not final bridge-specific measures.

### 4. Shell and local-group definitions are still operational
The current shell and local-group labels are useful, but they are not yet fully canonically interpreted physical structures.

For all these reasons, the matrix must be read as a structured indication, not a resolved conclusion.

## Strongest current internal takeaways

The following internal takeaways are currently justified.

### A. Weighted arrangement matters
The current bridge-facing response depends not only on weight values, but on their relational placement.

### B. Global disruption hits hardest
This is expected and confirms that full arrangement is structurally important.

### C. Shell-preserving disruption is partially buffered
This suggests that shell structure may already carry part of the relevant order.

### D. Local-group disruption is surprisingly potent
This points toward strong local structural sensitivity inside the weighted relational layer.

### E. The bridge candidate likely has internal substructure
The weighted relational layer should no longer be treated as a single undifferentiated block.

## Recommended next steps

The matrix points directly to several follow-up steps.

### 1. Clean the matrix runner output
The current matrix summary still includes noisy lines from the per-run stdout stream. This should be fixed so the summary remains clean.

### 2. Run the same matrix on a real extracted baseline
This is essential before taking the current comparative pattern too far.

### 3. Add shell-crossing perturbation tests
A direct comparison between:
- shell-preserving reweighting
and
- shell-breaking reweighting

would be one of the strongest immediate follow-up tests.

### 4. Add stronger shell-specific readouts
The shell signal now deserves its own dedicated readout family.

### 5. Clarify local-group construction
Because the local-group effect appears early, the construction of `local_group` becomes methodologically important.

## Bottom line

The first BMC-01 matrix does not yet prove that the weighted relational layer is a full bridge carrier.

But it does show something important:

- the arrangement of weights matters
- local reorganization matters
- global disruption matters most
- and shell-preserving perturbation appears comparatively buffered at lower strengths

This makes the weighted relational layer look increasingly like a **structured internal order field**, not merely a flat diagnostic surface.

The strongest current internal consequence is this:

> the bridge candidate now appears internally layered, with shell structure and local weighted arrangement emerging as distinct and potentially meaningful sublayers of order.
