# BMS-FU01 — Runner Field List

Date: 2026-05-01  
Project: Gravitation und RaumZeit / Quantum-Spacetime-Bridge  
Recommended repo target: `docs/BMS_FU01_RUNNER_FIELD_LIST.md`  
Status: Companion field list for BMS-FU01 C60 structure-information diagnostic

---

## 1. Purpose

BMS-FU01 applies a BMC-15h/ST01-style structured-specificity diagnostic to the audited C60 / truncated-icosahedral graph generated by:

```text
scripts/build_bms_fu01_c60_graph.py
```

The run tests whether the diagnostic distinguishes:

```text
degree-3 graph regularity
edge-class placement
local ring-motif structure
controlled core-seeded reproduction
```

| field name | type | description |
|---|---:|---|
| `degree_preserving_rewire` | null family | Preserves degree sequence while perturbing topology and shuffling edge-class counts. |
| `edge_class_shuffle` | null family | Preserves topology and edge-class counts but shuffles edge-class placement. |
