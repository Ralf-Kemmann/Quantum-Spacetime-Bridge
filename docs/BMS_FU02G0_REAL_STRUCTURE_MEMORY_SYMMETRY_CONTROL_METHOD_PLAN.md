## 3. Real-structure memory

### 3.1 Working definition

```text
Real-structure memory denotes the extent to which a derived relational or
carrier structure retains recoverable information about a known reference
structure beyond trivial graph size, degree, edge-count, local-connectivity or
boundary effects.
```

German internal version:

```text
Realstruktur-Gedächtnis ist das,
was von der echten Struktur in der Korrelationsstruktur wiederauffindbar bleibt,
nachdem triviale Graph-Effekte kontrolliert wurden.
```

### 3.2 Memory analogy

```text
The reference structure is the stored content.
The relational/correlation structure is a partial, indirect and possibly noisy
trace of that content.
The diagnostic asks what can be reconstructed from the trace.
```

Project-internal image:

```text
Die Realstruktur ist der gespeicherte Inhalt.
Die Korrelationsstruktur ist eine teilweise Spur davon.
Die Diagnostik fragt:
  Was lässt sich daraus wiederherstellen?
```

## 7. Key methodological issue: local patch vs symmetry orbit

In highly symmetric structures, a localized carrier patch should not be
interpreted too quickly as a unique physical location.

For C60, the carrier region may be:

```text
a specific unique defect-like region,
a representative of a symmetry-equivalent patch family,
or a construction-selected local patch in an otherwise symmetric graph.
```

Important principle:

```text
If a reconstructed patch is mapped by automorphisms to many equivalent patches,
then the physically/mathematically relevant object is the patch class, not the
specific indexed location.
```

Project-internal formulation:

```text
Nicht der konkrete Ort muss spezifisch sein.
Spezifisch kann die Strukturklasse des Klunkers sein.
```

structures.
