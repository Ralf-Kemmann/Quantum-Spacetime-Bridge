# Gravitation und RaumZeit (GRZ) — Project Scaffold v1

This folder structure is designed to keep the project reproducible, AI-friendly, and submission-ready.

## Quick start
1) Put the master narrative / curated rules into `docs/masterchat.md`
2) Log every decision/experiment/submission in `docs/project_log.md` (60 seconds per entry)
3) Keep journal submissions separate under `manuscripts/` and `submissions/`
4) Keep code under `code/` and any raw/sim outputs under `data/` / `outputs/`

## Suggested Git usage (optional)
- Initialize git in this folder
- Commit `docs/` frequently (it is your “single source of truth”)
- Tag submission snapshots (e.g. `grg-submission-v0.1`)

## Core rules (short)
- Separate: Definition / Derivation / Interpretation
- Use Common Support S(δ) + normalize on S(δ) (restrict then normalize)
- Treat Newton/geometry as a *limit diagnostic*, not a postulate
