# debroglie-phase-bridge

Mechanismus-Demonstrator für den Projektkern

de Broglie → Phasenstruktur → Interferenz → Korrelation → Strukturgraph → D_rel

## Status

Dieses Repository enthält den aktuellen Projektstandard als nachvollziehbare und reproduzierbare Arbeitsbasis.

Aktuelle Schwerpunkte:

- K.0 / N.0: Referenzimplementierung und Null-Modell
- T.1 / T.1a / T.1b: kritische-θ-Karte, D_rel-Auslese, Parameterfenster
- M.1: komponentenweise metrische / pseudometrische Einordnung

## Wichtige Einordnung

Dies ist **kein fertiges Raumzeitmodell**, sondern ein **Mechanismus-Demonstrator** mit defensivem Claim-Niveau.

## Repo-Struktur

- `docs/` — Theorie, Projektregeln, Review-Antworten
- `src/dpb/` — kanonischer Code
- `scripts/` — reproduzierbare Einstiegsskripte
- `configs/` — Laufkonfigurationen
- `results/` — erzeugte Resultate
- `tests/` — Regression und Referenztests

## Erster Schritt

Die Referenzimplementierung wird in `src/dpb/` aufgebaut. Einstiegsskripte liegen in `scripts/`.
