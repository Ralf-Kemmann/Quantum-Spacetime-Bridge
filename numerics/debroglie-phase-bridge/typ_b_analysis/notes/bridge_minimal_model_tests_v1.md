## 2. Erste empirische Minimalerwartung aus `bridge_minimal_model_v1`

Für die derzeit signalhaltigen Datensätze erwartet das Modell:

- `negative` und `abs` bilden einen ähnlichen Brückenzustand
- `positive` bleibt systematisch schwächer
- `phase_influence` ist für `negative/abs` klein, aber nicht null
- `phase_influence` ist für `positive` im Makroregime praktisch null
