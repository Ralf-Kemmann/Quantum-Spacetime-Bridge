## 3. What is currently a strong but still open mechanism hypothesis
The following points are not random speculation.

### 3.1 Bindable relational overlap
The project currently has a strong reason to distinguish:
- generic overlap
from
- bindable overlap

This is conceptually strong, but not yet fully operationally isolated.

The current best internal reading is that a compatibility layer is required between:
- overlap
and
- stabilization / form

This layer would answer:
- which overlaps are actually jointly supportable
- which ones are not

This is probably the most important missing mechanism piece in the current block.

At present:
- conceptually necessary

This is the main open mechanism gap.

### 3.3 Bound transition structure
The bridge is currently best read not as:
- a simple connector
- or a mixed blur

but as:
- a structured, internally organized bound transition structure

This is a strong conceptual advance.

However, it still needs:
- clearer falsification logic
- and eventually a more explicit operational test for when the bridge deserves this label and when it does not

So it is strong, but not yet final.

---

### 3.4 Readability layer as mechanism-relevant distinction
The readability layer is not just philosophical decoration.
It currently functions as a mechanism-relevant distinction:

- deep organization may come first
- readable regime / geometry / appearance may come later

This idea is strong and useful.
But the project still lacks a direct diagnostic case in which:
- organization is visibly already present
- while readability is still clearly absent

That remains an important open target.
